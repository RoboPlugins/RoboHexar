Robohexar

I am a IntelliJ Plugin that pre-reviews your code before it goes before the Hexar for judgment. I have been activated. Missing punctuation and grammar will be punished.

View on GitHub Download .zip Download .tar.gz

Installation

From within an Idea IDE, choose:

  1. Settings.
  2. Plugins.
  3. Browse Repositories.
  4. Search for "Robo Hexar".
  5. Profit.

CI Build Installation (Frequent changes)

  1. Settings.
  2. Plugins.
  3. Browse Repositories.
  4. Manage Repositories.
  5. Add https://roboplugins.github.io/updatePlugins.xml
  6. Profit.
